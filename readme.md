<h1 align="center">Gallus Gallus Monogatari</h2>

### Description du produit final
Incarnez un poulet samouraï, astronaute à temps partiel, à la fois redoutable et adorable, et protégez vos poussins contre des animaux prédateurs dans une aventure épique pour sauver votre village.

### Processus d'initialisation
1. S'assurer d'avoir la **dernière version** d'Unity d'installé (Unity 3.16.2).
2. cloner le projet à l'emplacement de votre choix.
3. Dans Unity, ouvrir un projet et choisir le dossier du jeu.
4. build le jeu.
5. Enjoy!
